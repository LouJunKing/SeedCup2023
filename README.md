## 2023种子杯-鹤冲天队伍项目

赛题详见[seedcup2023-client](https://github.com/Thyme-git/seedcup2023-client)

算法介绍详见[答辩ppt](./鹤冲天_种子杯决赛答辩.pptx)

### 运行

先在bin目录下启动服务器
```bash
cd bin
./server
```

然后添加第一个玩家/电脑，比如silly-bot
```bash
cd bin
./silly-bot
```

最后在player目录下启动
```bash
cd player
python3 Player.py
```

请确保环境中已安装numpy
